Review Flashcards

What does HTML stand for?
HyperText Markup Language
What does CSS stand for?
Cascading Style Sheets
What does JS stand for?
JavaScript
How do you link a CSS file to an HTML document?
By using the '<link>' tag inside the header, '<link rel="stylesheet" href="style.css">'.
What is the difference between '<ul>' and '<ol>' in HTML?
The '<ul>' tag creates an unordered list with bullet points, while the '<ol>' tag creates an ordered list with numbered items.
What does the CSS "margin" property do?
The CSS margin property controls the space that is outside of the border of an element, creating space between the affected element and neighboring ones.
How to add a comment in HTML?
To add a comment in HTML, you must wrap the text in '<!--' and '-->'. For example: '<!-- This is a comment -->'
What does URL stand for?
URL stands for Uniform Resource Locator
